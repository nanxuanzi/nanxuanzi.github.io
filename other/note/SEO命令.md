# SEO命令

> ## 搜索引擎命令

|    命令    |             解释             |
| :--------: | :--------------------------: |
|  “双引号”  |         完全匹配搜索         |
|     -      |            不匹配            |
|            |                              |
|    site    | 搜索某个域名下收录的所有文件 |
|   inurl    |   返回url包含关键词的页面    |
|  allinurl  |   url内同时包含多个关键字    |
|  intitle   | 页面title中包含关键词的页面  |
| allintitle |                              |
|            |                              |

> ## 实例

1. 完全匹配

   搜索结果返回的页面包含： ”南轩子“

   ````SEO
   "南轩子"
   ````

2. 不匹配

   搜索返回的页面包含：搜索，但是屏蔽：引擎

   ```
   搜索 -引擎
   ```
