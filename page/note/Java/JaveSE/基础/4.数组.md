# 数组

## 声明

```java
//静态初始化
int[] array1 = {1,2,3,4}

//动态初始化, 指定数组长度，由系统分配初始值
int[] array2 = new int[5]
```

## 常见操作

```java
//存取
array2[0] = 1;
array2[1] = 2;
array2[2] = 3;

//length
array2.length
  
//循环遍历数组
for (int i = 0; i < array4.length; i++) {
   System.out.printf("array[%d]的值：%d；\n",i,array4[i]);
}

//For-Each增强循环，无下标
//（数组内的元素 : 数组）
for(int arr:array4){
    System.out.println(arr);

}
```
