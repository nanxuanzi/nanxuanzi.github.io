# 条件结构

## if

```java
//if...else if...else
public class Test {
    public static void main(String[] args) {
                int a = 1;
                int b = 2;
                if(a==b){
                    System.out.println("a等于b");
                } else if(a>b){
                    System.out.println("a大于b");
                } else{
                    System.out.println("a小于b");
                }
    }
}
```

## switch

```java
//switch..case 
public class Test{
    public static void main(String[] args){
        int a = 2;
        switch(a){
            case 1:
                System.out.println("a等于1");
                break;
            case 2:
                System.out.println("a等于2");
                break;
            case 3:
                System.out.println("a等于3");
                break;
        }
        
    }
}
```
