# 循环结构

## for

```java
public class Test{
    public static void main(String[] args){
        for (int i = 0; i < 5; i++) {
            System.out.println(i);
        }
    }
}
```



## while

```java
//while(true)，时执行{}
public class Test{
    public static void main(String[] args){
        int i = 0;
       while (i<5){
           System.out.println(i);
           i++;
       }
    }
}
```



## do...while

```java
//do while至少执行一次
public class Test{
    public static void main(String[] args){
        int i = 0;
        do {
            System.out.println(i);
            i++;
        } while (i<5);
    }
}
```
