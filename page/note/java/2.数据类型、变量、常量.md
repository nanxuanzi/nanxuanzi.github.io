# 数据类型

## 基本

```java
//字符、字符串
char a = 'a';
String b = "bbbb";

//数字：int、float、double
int c = 3;
float d = 4.00F;
double e = 4.00;

//布尔：boolean
boolean f = false;// true|false
```

## 变量

1. 类变量（静态变量）：独立于方法之外的变量，用static修饰
2. 实例变量：独立于方法之外的变量，不用static修饰
3. 局部变量：类的方法中的变量

## 常量

常量一般用final修饰

```java
//常量一般用大写字符
final NAME = "nanxuanzi";
```
